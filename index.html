Data Analysis Dashboard: Salaries In Data

In this project, I created a visually asthetic dashboard in Power BI to analyze the different salaries in the field of Data for the period of 2020-2023. The dataset consists of salaries in USD for every country in the world which I placed into a slicer to narrow the analysis down for each country. The dashboard is fully interactive using the Year, Country, Work Setting and Job Title slicers.

Data Analysis Dashboard: Global Video Game Sales

In this project, I created a video game sales dashboard that has data recorded from 1980 till 2020. I split the regions into Europe, Japan and USA Sales as these 3 countries have been the top producers of video games and I have incorporated all the other countries into a seperate graph to have more accurate readings of the data. The dashboard is fully interactive with the Platform, Genre and Year slicers.
